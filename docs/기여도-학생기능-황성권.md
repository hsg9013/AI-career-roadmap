| 4 | **활동 기록 → 문서 자동 생성** | 황성권 | 이력서·자기소개서·포트폴리오 생성 버튼 + **PDF·Word 실제 다운로드** + draft/final 상태 관리 구성(화면 캡처가 아닌 실제 파일 생성) | 「TO-BE 목표 기능」 · 「갭 분석」 **H5** · 「TO-BE 앱·기술 아키텍처」 · ISP §7·§9 |
